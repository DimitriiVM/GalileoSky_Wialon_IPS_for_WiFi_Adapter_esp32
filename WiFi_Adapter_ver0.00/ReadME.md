Описание функций для WiFi адаптера версия 0.00

RS485 припаимается RX к 16 пину, TX к 17 пину.
Скорость RS485 - 19200
Скорость USB - 115200
Размер массива на  отправку не должен превышать 1024 байта

Все команды подджерживаются как через USB так и через RS485.
Для  управления  по USB воспользуйтесь программой: Terminal 1.9b

Команды:

RESET - Перезагрузка модуля
ОТВЕТ: OK\r\n

AT+WFCONN - Подключения к WiFi
ОТВЕТ: OK\r\n
+WFCONN:OK\r\n or +WFCONN:ERROR\r\n

AT+WFDIS - Отключение от WiFi
ОТВЕТ: +WFDIS:OK\r\n

AT+SERVCONN\r\n - Подключение к серверу
ОТВЕТ: +SERVCONN:OK\r\n or +SERVCONN:ERROR\r\n

AT+SERVDIS\r\n - Отключение от сервера
ОТВЕТ: +SERVDIS:OK\r\n

AT+SEND=ДАННЫЕ\r\n - отправка данных на сервер по WiFi
ОТВЕТ: +SEND:OK\r\n or +SEND:ERROR\r\n

Если сервер пришлет какие  либо  данные,  то они  будут автоматически отправлены в RS485 без каких либо признаков, что это  данные с сервера

AT+PORT=ПОРТ\r\n - Установка порта сервера
ОТВЕТ: +PORT:ПОРТ\r\n

AT+HOST=DNS or IP\r\n - Устанвока адреса сервера куда подключатся
ОТВЕТ: +HOST:DNS or IP\r\n

AT+SSID=Назвение WiFi сети\r\n - Точка для подключения
ОТВЕТ: +SSID:Назвение WiFi сети\r\n

AT+PASS=Пароль от WiFI\r\n - Пароль от точки подключения
ОТВЕТ:+PASS:Пароль от WiFI\r\n

AT+\r\n - Команда проверки на связи ли модуль
ОТВЕТ: OK\r\n
